# Product page playbooks

This folder is a **practical library** of focused notes for Shopify merchants and theme developers using **Product Info Advanced** and **Product Announcement Bar**.

Each playbook is intentionally short: one topic, clear steps, and a checklist you can apply in the Theme Editor or in Liquid metafields, without a third-party app layer.

## How to use

1. Pick a topic that matches your current risk (SEO, subscriptions, accessibility, performance, cross-border, etc.).
2. Apply the steps in staging or a duplicate theme first.
3. Map recommendations to your section blocks (bundle cards, trust signals, subscription toggle, media carousel).
4. Measure for at least one full business cycle before stacking more changes.
