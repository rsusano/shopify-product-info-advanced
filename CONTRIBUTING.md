# Contributing to Product Info Advanced

Thank you for your interest in contributing! This document outlines how to submit changes.

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change: `git checkout -b fix/your-feature-name`
3. **Make your changes** and test locally
4. **Commit** with a clear message: `git commit -m "Add: brief description"`
5. **Push** and open a Pull Request: `git push origin fix/your-feature-name`

## Pull Request Guidelines

- Keep PRs focused on a single change
- Ensure the theme works in the Shopify Theme Editor
- Update the README or INSTALL.md if you change behavior

## Reporting Issues

Use the [GitHub Issues](https://github.com/rsusano/shopify-product-info-advanced/issues) page. Include:

- Shopify theme name and version
- Steps to reproduce
- Expected vs actual behavior
